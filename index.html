HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • html elements work together to build the structure of a webpage and provide a framework that makes it easy to navigate
  • elements are composed of an opening tag and a closing tag represented by "< and >"

CSS

  • Class atribute allows us to share a CSS rule to any element we choose by assigning the rule to a class atribute with a class selector, then the class atribute can beassigned to the element in html, allowing us to freely choose which elements will be assigned the css rule.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
